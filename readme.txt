TINET.VN Chat Buttons is a free WordPress plugin for adding the live chat/call functionality of popular social media messaging apps.

== Description ==

The "TINET Chat Buttons" displayes an fixed contact button on the bottom right of the screen. It opens an modal with your favorite contact form plugin. Before you have to generate a contact form and after this copy the shortcode into the plugin settings form Floating Contact Button.
The plugin is real lightweight. The .css file is minimized to load as fast as can.

== Frequently Asked Questions ==

= Can i change the style from the insert plugin shortcode? =

Optical adjustments of the used contact plugin have to be done user-defined e.g. in the theme own style.css.
